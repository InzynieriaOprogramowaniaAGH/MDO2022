Sprawozdanie

1.	Zainstalowannie gita na Ubuntu na WSL

2.	Wygenerowanie dwóch kluczy ed25519 (jeden z hasłem, drugi bez) potrzebnych do połączenia się z GitHubem za pomocą SSH za pomocą komendy >ssh-keygen -t ed25519
Oba klucze znajdują się w katalogu .ssh.

3.	Przejście do odpowiedniego folderu i sklonowanie repozytorium MDO2022 za pomocą http, komendą >git clone https://github.com/InzynieriaOprogramowaniaAGH/MDO2022.git

4.	Przejście do innego pustego katalogu i sklonowanie repozytorium MDO2022 przez SSH, z użyciem wybranego klucza z hasłem za pomocą komendy
>GIT_SSH_COMMAND="ssh -i ~/.ssh/id-ed25519" git clone git@github.com:InzynieriaOprogramowaniaAGH/MDO2022.git

5.	Przełączenie na gałąź odpowiedniej grupy GCL02

6.	Stworzenie własnej gałęzi EP307681

7.	Stworzenie własnego katalogu EP307681

8.	Dodanie do katalogu pliku ze sprawozdaniem i zrzutów ekranu

9.	Zaznaczenie wszystkich plików >git add .

10.	Stworzenie nowego commita > git commit -m lab1.1

11.	Wysłanie zmian do zdalnego źródła, w którym jeszcze nie jest stworzona nowa gałąź
> git push --set-upstream origin EP307681

12.	Przejście z powrotem do brancha GCL02 > git checkout GCL02

13.	Zmergowanie gałęzi EP307681 do GCL02

14.	Zaznaczenie plików, stworzenie commita i próba wysłania zmian do zdalnego źródła, które zakończyło się niepowodzeniem, ponieważ wymagana jest przynajmniej jedna recenzja

15.	Oznaczenie tagiem „v1.0.0.1” ostatniego commita „b60dca4” 
> git tag -a v1.0.0.1 b60dca4 -m wlasny-tag

16.	Wysłanie zmian z tagami do zdalnego źródła
> git push --tags

17.	Stworzenie git hook’a, który sprawdza czy w wiadomości commita znajduje się skrót nazwy przedmiotu „MDO”. Hooki znajdują się w folderze repozytorium w .git/hooks.

#!/bin/bash

MSG="$1"

if ! grep -qE "MDO" "$MSG";then
    cat "$MSG"
    echo "Ten commit nie zawiera nazwy przedmiotu MDO"
    exit 1
fi

18. Aby stworzyć hooka, który automatycznie dopisuje nazwę przedmiotu należy w poprzednim programie dodać zmianę wartości zmiennej $MSG, czyli wiadomości commita.
